Easy Backup Plugin for WordPress to create, download and restore backups of your WordPress website.

== Description ==

**WP Backup Bank** is a plugin that provides backup capabilities for WordPress. Backups are zip archives for WordPress and .sql for Database Backups.

[Backup Bank](http://tech-banker.com/products/wp-backup-bank/)

= Features =

* Manual backups.
* Back up the database.
* Back up files and directories.
* Fine grained control over what gets backed up.
* Store backups locally.
    
= Technical support =

Please email all your queries/issues at support@tech-banker.com

== Installation ==

1. Upload the `wp-backup-bank` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.
